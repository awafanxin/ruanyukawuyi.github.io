下载东西

群文件”玩星铁必须安装的东西”里下载服务端需要安装的环境和Mongodb数据库和服务端:

下载与安装

安装刚刚下载的数据库和环境
下载客户端:http://j1.pw/sfw8QM

解压客户端和服务端

这就是解压后的样子

服务器配置，启动以及代理

打开服务器文件夹
打开一键安装凭证和npm

这个样子就是安装成功了

服务器启动

打开“一键启动”

这个样子就是启动成功

代理

打开“一键启动”的时候应该会自动打开fiddler
找到Tools点Options
点开HTTPS，把“Decrypt HTTPS traffic”点上

再点到“ Connections”，把下面的888改成10888

回到“Rules”点“Cuttomize Rules…” （打开时会弹出个选项，点确定）
里面会有一串东西，全删掉
import System;
import System.Windows.Forms;
import Fiddler;
import System.Text.RegularExpressions;

class Handlers
{
static function OnBeforeRequest(oS: Session) {
if(oS.host.EndsWith(“.yuanshen.com”) || oS.host.EndsWith(“.hoyoverse.com”) || oS.host.EndsWith(“.mihoyo.com”) || oS.host.EndsWith(“.starrails.com”)) {
oS.host = “localhost”;
}
}
};
把这些东西复制进去,保存就行了

创建账号与游玩

在刚刚点“一键启动”打开的cmd窗口哪里
输入 account create 用户名 uid (比如：accont create 123 123456)
之后就可以直接进入刚才解压的客户端文件夹里面打开游戏了

如果出现这个东西点Yes即可

指令

指令都在刚刚注册的cmd窗口输入
输入命令前先绑定uid：target <你的uid>
注意绑定前，角色必须在线!!
具体指令可以去看群文件
地图 ID: 在 CrepeSR\src\data\excel 中的 MapEntryExcelTable.json 里可以找到能用来改编场景的 planeID 和 floorID 。
改变地图: 在 CrepeSR\src\server\packets 中的 GetCurSceneInfoCsReq.ts 里编辑 planeId 和 floorId 来改变场景。
